Lane Detection in OpenCV. To exit the program press "q".

## Installation

```bash
pip install -r requirements.txt
```

## Requirements

- [OpenCV](https://pypi.org/project/opencv-python/) Version 4.2.0
- [Numpy](https://pypi.org/project/numpy/) Version 1.18.4
